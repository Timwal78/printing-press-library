# AGENTS.md — ghost-layer-pp-cli

Agent operating guide for the Ghost Layer CLI. Generated by CLI Printing Press.

## Identity

- **Binary**: `ghost-layer-pp-cli`
- **API**: Ghost Layer — proprietary dual-chain XRPL/Base toll gateway
- **Base URL**: `https://ghost-layer.onrender.com`
- **Auth**: None for read endpoints. Bridge execution requires XRPL signature or EIP-3009.

## Commands

| Command | Description |
|---------|-------------|
| `ghost-layer status` | Health check — confirm service is live |
| `ghost-layer x402 catalog` | List all purchasable x402 products |
| `ghost-layer x402 quote --product <id> --wallet <addr>` | Get payment quote + XRPL destination |
| `ghost-layer x402 dispense <product_id>` | Dispense product post-payment |
| `ghost-layer bridge --chain XRPL\|Base ...` | Execute a cross-chain settlement |
| `ghost-layer agent <wallet>` | Agent stats, loyalty tier, Passport level |
| `ghost-layer cube state` | 54-block execution matrix snapshot |

## Recommended Agent Flow

```
1. ghost-layer status                          # confirm live
2. ghost-layer x402 catalog                    # see available products
3. ghost-layer x402 quote --product routing.telemetry --wallet rXXX
   # → returns pay_to address + memo_hex + amount
4. # Pay on XRPL: send RLUSD to pay_to with memo_hex as MemoData
5. ghost-layer x402 dispense routing.telemetry # after payment confirmed
6. ghost-layer agent rXXX                      # check loyalty tier earned
```

## Environment Variables

| Variable | Default | Purpose |
|----------|---------|----------|

| `GHOST_LAYER_BASE_URL` | `https://ghost-layer.onrender.com` | Override base URL |

## Architecture Note

Ghost Layer is the proprietary foundation that three products run on:
1. **Agent Kiosk** — AI agent vending machine (X-Grace-Tip auction)
2. **TipMaster** — Farcaster RLUSD tip bot
3. **Signal Auction Loom** — real-time WebGL auction visualizer

All three are licensed through Ghost Layer's X402-BEAST-KEY URIToken system.
